Formations

Master 2 3IR 2021/2022

Cette formation propose des enseignements de préparation à l’insertion professionnelle. Elle conserve un tronc commun important et propose une spécialisation via ses deux parcours : le parcours ” images” s’oriente vers le domaine du traitement et transport des contenus audio-visuels ; le parcours “réseaux” s’oriente vers les technologies réseaux et le développement d’applications Internet.

Spécialités:

  • Application Web
    • HTML
    • CSS
    • JavaScript
  • Applications Internet Réparties
    • Java EE
    • Maven

Master 1 3IR 2020/2021

Cette formation vise à donner aux étudiants les bases scientifiques d’une compétence en traitement du signal et de l’image, en réseaux informatiques, en technologie Java, ainsi qu’en télécommunications numérique.

Spécialités:

  • Protocoles Internet
    • Protocoles de Routages: RIPv2, OSPF
    • TCP/IP
    • Adressage
  • Jave pour les réseaux et les images
    • Dijkstra
    • Structures de données
